# 01 · Терминал и Homebrew — фундамент

## Зачем это
Терминал — это твой **пульт управления** Mac. А Homebrew — **магазин** маленьких инструментов для этого пульта: вместо того чтобы искать программы по интернету, ты говоришь «поставь мне вот это» — и оно ставится само. Сегодня ставим магазин и проверяем, что Mac тебя слышит.

> Аналогия: Терминал — пульт от телевизора. Homebrew — App Store, только для маленьких пультовых команд.

## Шаги по порядку

### Шаг 1. Поставить Homebrew (и инструменты Apple)
**Что произойдёт:** Mac сначала может попросить поставить «инструменты командной строки» Apple (маленькое окно с кнопкой **Установить** — нажми её). Потом установится сам Homebrew. **Тебя попросят пароль от Mac** — это нормально; символы пароля не видны, просто введи и нажми Enter.

Волшебная кнопка:
```bash
bash 02-foundation/install-homebrew.sh
```

**Что ты увидишь:** много строк (Homebrew любит подробно рассказывать, что делает), а в конце — зелёное `✅ Homebrew работает: Homebrew 4.x.x`.

### Шаг 2. Поставить первый инструмент — `tree`
**Что произойдёт:** через магазин Homebrew поставится маленькая утилита `tree`. Это «доказательство жизни»: раз поставилось — магазин работает.

Волшебная кнопка:
```bash
bash 02-foundation/install-tree.sh
```

**Что ты увидишь:** `✅ Сработало: tree v2.x.x`.

### Шаг 3. Проверить, что всё на месте
```bash
bash 02-foundation/verify.sh
```
**Что ты увидишь:** две зелёные строки — Homebrew и tree.

## Чек-поинт фазы
Фаза пройдена, если:
- `brew --version` показывает номер версии;
- `tree --version` показывает номер версии;
- `verify.sh` закончился словами «Фаза 02 пройдена».
