# 01 · Настройка MacBook — готовим рабочее место

## Зачем это
С этого начинается всё. Прежде чем ставить инструменты, **настроим сам Mac** так, чтобы тебе было удобно и безопасно работать: чтобы были видны нужные файлы, Mac не засыпал посреди долгой задачи, клавиатура не «исправляла» твой код, а экран был защищён. Это разовая настройка — сделал и забыл.

> Аналогия: прежде чем готовить, мы наводим порядок на кухне — раскладываем доски, точим ножи, включаем свет.

## Шаги по порядку

### Шаг 1. Настроить систему (одна волшебная кнопка)
**Что произойдёт:** скрипт настроит Finder (покажет скрытые файлы вроде `.env` и `.git`, полный путь, все расширения), режим сна (Mac не уснёт при работе от сети), клавиатуру (быстрый отклик, без автозамены кавычек), безопасность (пароль сразу после сна, файрвол), Dock и скриншоты. **В середине попросит пароль от Mac** — для настройки сна и файрвола. Символы пароля не видны, это нормально.

Волшебная кнопка:
```bash
bash 01-macbook-setup/setup-mac-defaults.sh
```
**Что ты увидишь:** блоки `✅ настроено`, в конце — `🎉 MacBook настроен под вайбкодинг`. Finder и Dock на миг моргнут — так применяются настройки.

### Шаг 2. Проверить
```bash
bash 01-macbook-setup/verify.sh
```
**Увидишь:** зелёные галочки по ключевым настройкам.

### Шаг 3 (по желанию). Удобные приложения
**Что произойдёт:** можно поставить Cursor, VS Code, iTerm2, Warp, Raycast, Rectangle. Это **не обязательно** — курс работает в Claude Code и Codex. Скрипту нужен Homebrew, поэтому запускай его **после фазы 02** (где ставится Homebrew), либо пропусти совсем.
```bash
bash 01-macbook-setup/install-extra-apps.sh
```

## Что включить вручную (важно, скрипт намеренно НЕ делает это сам)
Эти две вещи касаются безопасности и требуют твоего решения — включи их руками:

- **FileVault (шифрование диска).** `Системные настройки → Конфиденциальность и безопасность → FileVault → Включить`. Если Mac украдут — твои файлы и ключи не прочитают.
- **Touch ID для `sudo`** (по желанию) — чтобы в Терминале прикладывать палец вместо пароля. Включается правкой системного файла; делай это, только если понимаешь шаг (подробности — в инструкции Apple/курса). Если сомневаешься — пропусти, пароль работает и так.

## Чек-поинт фазы
Фаза пройдена, если:
- `setup-mac-defaults.sh` дошёл до `🎉 MacBook настроен`;
- `verify.sh` показывает зелёные галочки (скрытые файлы, путь в Finder, расширения, автоскрытие Dock, пароль после сна);
- ты решил по FileVault (включил или осознанно отложил).
