# 03 · Два ИИ-помощника — нанимаем исполнителей

## Зачем это
Ты становишься **режиссёром**, а не программистом, который пишет каждую строчку. Сегодня нанимаем двух исполнителей. **Claude Code** — главный помощник курса: читает проект, меняет файлы, объясняет ошибки. **Codex** — второй помощник и второй взгляд: проверить план или предложить другой способ.

> Аналогия: Node — розетка, npm — курьер, ИИ-помощники — приборы, которые включаются в эту розетку.

## Шаги по порядку

### Шаг 1. Поставить Node и npm
**Что произойдёт:** через Homebrew поставится Node, а вместе с ним npm.
```bash
bash 04-ai-helpers/install-node.sh
```
**Увидишь:** версии Node и npm.

### Шаг 2. Поставить Claude Code
**Что произойдёт:** npm поставит команду `claude`. При **первом запуске** `claude` откроется браузер для входа — это делаешь ты сам.
```bash
bash 04-ai-helpers/install-claude.sh
```
**Увидишь:** `✅ Claude Code на месте` (или сообщение, что версия покажется после входа).

### Шаг 3. Поставить Codex
**Что произойдёт:** npm поставит команду `codex`. При первом запуске попросит войти через ChatGPT.
```bash
bash 04-ai-helpers/install-codex.sh
```

### Шаг 4. Познакомиться с помощниками (по желанию, но полезно)
Запусти главного помощника:
```bash
claude
```
И вставь ему дружелюбный первый промпт:
```text
Привет! Объясни мне простыми словами, что ты умеешь делать в проекте Flutter. Не меняй файлы. Дай 5 коротких пунктов.
```
Если вместо ответа открылся вход в аккаунт — это **хороший знак**: помощник установлен и ждёт задания. Войди и повтори.

### Шаг 5. Проверить установку
```bash
bash 04-ai-helpers/verify.sh
```

## Чек-поинт фазы
Фаза пройдена, если:
- `node -v` и `npm -v` показывают версии;
- команды `claude` и `codex` запускаются (пусть даже просят вход в аккаунт);
- `verify.sh` закончился «Фаза 04 пройдена».

## Что дальше
Переходи в **`05-flutter/`** — ставим фабрику приложений Flutter и мобильные «цеха».

> 🚫 Главное правило обоих помощников: **один промпт — одна задача**. Не проси «сделай всё приложение». Проси «измени заголовок в одном файле».
> 🚫 Не давай ИИ доступ к секретам, паролям и токенам. Не разрешай необратимое (удаление, оплата, деплой, merge) вслепую — только после явного «да».
