# 05 · Flutter и мобильные цеха — фабрика приложений

## Зачем это
Ставим **фабрику приложений** — Flutter. Из одного проекта она делает приложения для iPhone, Android и компьютера. Но станку нужны **цеха**: **Xcode** (для iPhone, от Apple), **Android Studio** (для Android, от Google) и **CocoaPods** (менеджер деталей для iOS). А ещё **игрушечный телефон** — симулятор iPhone на экране Mac.

> Самая важная команда фазы — `flutter doctor`. Это не «починить всё сам», а **медицинская карта**: что готово, чего не хватает. Красные крестики — это список дел, **не поломка**.

## Шаги по порядку

> ⏳ Эта фаза — самая долгая (большие загрузки) и единственная, где будет **пароль от Mac** и установка из **App Store**. Скрипты предупреждают перед каждым таким моментом и останавливаются.

### Шаг 1. Поставить Flutter
```bash
bash 05-flutter/install-flutter.sh
```
**Увидишь:** версию Flutter.

### Шаг 2. Настроить Xcode (цех для iPhone)
**Что произойдёт:** если Xcode ещё нет — скрипт остановится и попросит установить его **из App Store** (несколько ГБ, ставится долго). Потом подключит Xcode к системе — здесь будет **пароль от Mac** (`sudo`) и принятие лицензии Apple, затем докачается iOS-платформа.
```bash
bash 05-flutter/install-xcode-tools.sh
```

### Шаг 3. Поставить Android Studio (цех для Android)
**Что произойдёт:** поставится Android Studio, потом скрипт попросит **открыть её один раз** и пройти Setup Wizard (он докачает Android SDK). После — принятие лицензий Android (отвечай `y`).
```bash
bash 05-flutter/install-android-studio.sh
```

### Шаг 4. Поставить CocoaPods
```bash
bash 05-flutter/install-cocoapods.sh
```
**Увидишь:** версию CocoaPods.

### Шаг 5. Осмотр доктором
```bash
bash 05-flutter/verify.sh
```
**Что ты увидишь:** карту `flutter doctor`. Если есть красные пункты — **скопируй весь вывод** и отдай Claude Code или Codex с промптом:
```text
Я новичок. Вот полный вывод flutter doctor. Объясни простыми словами, что готово, что нет, и дай только ОДИН следующий безопасный шаг. Не предлагай сразу много действий.

[вставь сюда весь вывод flutter doctor]
```

## Чек-поинт фазы
Фаза пройдена, если:
- `flutter --version` и `pod --version` показывают версии;
- `flutter doctor` **запускается** и выдаёт карту (пусть даже не все галочки зелёные);
- Xcode установлен (для iPhone-симулятора).
